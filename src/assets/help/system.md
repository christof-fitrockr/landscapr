# System

The System section serves as an inventory of your Application Landscape. It includes all software applications, services, and platforms.

## System Editor
- **Base Info:** Name, Description, and Owner of the system.
- **APIs:** List the APIs (Functions) that this system exposes. You can create new API definitions directly from here.
- **Implements:** Map the system to the **Business Capabilities** it supports.
- **Usage:** See which **Processes** rely on this system (derived from the "Implemented By" field on Process steps).

## Application Portfolio Management
Use this section to keep track of your IT assets and their relationships to the business.
